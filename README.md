
# Forum

# Sobre
API desenvolvida para um desafio com Java Spring Boot.

# Iniciando
## Pré-requisitos
- `maven`

- `JDK`

## Instalação

- ### 1. Clonar o repositório
    `git clone https://github.com/felipenaf/forum`

- ### 2. Executar o projeto
	`mvn spring-boot:run`

## Observações
### Base de dados
